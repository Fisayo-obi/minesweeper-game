# Minesweeper Game (Java)

A console-based implementation of the classic Minesweeper game built in Java. The program allows users to select difficulty levels and interact with a dynamically generated game board.

## Features
- Multiple difficulty levels (Easy, Medium, Hard)
- Flagging system for marking suspected mines
- Win and loss condition handling
- Automatic reveal of adjacent empty cells
- Debug mode for visualizing mine placement

## Implementation Details
- Utilizes a 2D array to represent the game board
- Implements stack-based and queue-based traversal for revealing cells
- Handles user input and game state through an interactive loop

## How to Run
1. Compile all Java files
2. Run `Main.java`
3. Follow the prompts in the console to play

## Technologies
- Java
- Data structures: arrays, stacks, queues
